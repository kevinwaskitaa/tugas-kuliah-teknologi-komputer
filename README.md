# 🖥️ Portofolio Integrasi Teknologi & Produk Digital — D3 Teknologi Komputer

Selamat datang di repositori utama portofolio saya. Repositori ini dirancang khusus sebagai **One-Stop Showcase** (Portofolio Satu Pintu) yang mendokumentasikan seluruh rangkaian tugas kuliah, praktikum laboratorium, riset produk, hingga proyek implementasi koding murni yang saya selesaikan selama menempuh studi D3 Teknologi Komputer di Politeknik Internasional Tamansiswa Mojokerto.

Profil ini membuktikan kompetensi lintas disiplin saya dalam menggabungkan estetika antarmuka pengguna (**UI/UX Design**), riset berbasis data (**UX Research**), pemrograman aplikasi (**Frontend Web & Python Deep Learning**), serta pengelolaan infrastruktur sistem (**Linux Server & Jaringan MikroTik**).

## 🗂️ Peta Struktur Repositori Portofolio

Semua proyek di bawah ini dikelola secara terstruktur ke dalam subfolder independen yang masing-masing dilengkapi dengan dokumentasi teknis mendalam:

tugas-kuliah-teknologi-komputer/
│
├── 📁 01-vin-coffee-app/              # UI/UX Case Study & Interaktivitas Figma

├── 📁 05-deep-learning-python/        # Jupyter Notebook (.ipynb) & AI Komputasi Awan

🚀 Rincian Lengkap Proyek Portofolio
📁 Proyek 1: Case Study Vin Coffee & Cafe
Fokus Utama: UI/UX Design Thinking, User Flow, dan Interactive Prototyping.

Deskripsi Masalah: Sistem antrean manual di kasir kedai kopi sering kali memicu antrean panjang pada jam sibuk serta ketidakpastian ketersediaan stok menu bagi pelanggan di meja.

Solusi Digital: Merancang purwarupa (prototype) aplikasi pemesanan mandiri (self-ordering) berbasis mobile untuk memangkas waktu tunggu transaksi secara efisien.

Aset Portofolio: * 🔗 Akses Live Prototype Interaktif Figma (s.id/iwQbP)

🔗 Akses Moodboard Desain & Palet Warna (s.id/V3MJI)

📁 Proyek 2: Riset Produk Food Delivery Mahasiswa
Fokus Utama: UX Research, Analisis Data Kuantitatif & Kualitatif, dan User Journey Map.

Metodologi: Pengumpulan data primer melalui Google Forms yang disebarkan kepada 13 responden aktif dari kalangan mahasiswa.

Insight Kunci: Berdasarkan pengolahan data statistik, ditemukan fakta kritis bahwa 47% pengguna membatalkan pesanan di halaman checkout akibat biaya ongkos kirim yang terlalu mahal (mendapatkan skor rata-rata ketidakpuasan sebesar 3.08/5).

Output Visual: Menyusun matriks User Journey Map komprehensif mulai dari fase Trigger, Exploration, Checkout, hingga Usage untuk memetakan titik frustrasi (pain points) pengguna.

📁 Proyek 3: Wireframing Layout Antarmuka
Fokus Utama: Pembuatan Low-Fidelity Blueprint, Tata Letak Komponen UI, dan Evaluasi Heuristik.

Studi Kasus: Aplikasi pemesanan "Ayam Geprek Judes" untuk mengoptimasi alur kustomisasi produk (pemilihan level pedas dan menu tambahan).

Implementasi Elemen: Menerapkan kontrol UI yang presisi seperti Radio Button (opsi tunggal level pedas: Sopan, Sedang, Judes) dan Checkbox (opsi jamak untuk menu tambahan seperti nasi ekstra atau keju mozzarella) guna mencegah kesalahan input pengguna (error prevention).

Aset Portofolio: 🔗 Akses Dokumen Wireframe Figma (s.id/ukX8H)

📁 Proyek 4: Web Engineering & Lab Assignments
Fokus Utama: Frontend Development, Logika Pemrograman, dan Pengelolaan Git.

Penerapan Teknologi:

HTML5 Dasar: Penggunaan Semantic Tags, pembuatan tabel data terstruktur, serta form registrasi/login (Studi kasus: Artikel Sejarah Kota Mojokerto).

Modern CSS3: Implementasi sistem tata letak Flexbox dan Responsive Layout memanfaatkan media queries (@media) untuk optimasi tampilan mobile-first pada komponen Sidebar & Navigation Bar.

JavaScript DOM: Rekayasa manipulasi halaman web dinamis, Event Handlers, pembuatan aplikasi Counter, serta logika validasi form login di sisi klien (client-side validation).

Node.js Core Logic: Eksperimen algoritma dasar murni (seperti pemecahan kasus logika kelipatan angka FizzBuzz), penerapan OOP, serta manajemen kontrol versi repositori (version control).

📁 Proyek 5: Deep Learning & AI Computer Vision
Fokus Utama: Artificial Intelligence (AI), Machine Learning, Pemrograman Python, dan Ekosistem Cloud.

Implementasi: Pengembangan arsitektur jaringan saraf tiruan (Neural Networks) memanfaatkan pustaka berbasis Python untuk pengenalan pola data tingkat tinggi.

Fitur Integrasi: Seluruh skrip dikembangkan dalam format berkas .ipynb (Jupyter Notebook) di lingkungan Google Colab, terintegrasi langsung dengan GitHub melalui tombol otomatis "Open in Colab" untuk kemudahan pengujian komputasi awan.

📁 Proyek 6: Linux Server & Network Infrastructure
Fokus Utama: Administrasi Jaringan, Operasional OS Linux, Konfigurasi Server, dan Keamanan Sistem.

Cakupan Teknis:

Core Jaringan (MikroTik): Rancang bangun gateway internet, manajemen bandwidth menggunakan Simple Queue untuk mencegah bottleneck, serta penanganan wireless deployment (LAN/WLAN) di lapangan.

Sistem Operasi Server (Ubuntu Server): Instalasi dan konfigurasi layanan krusial seperti DHCP Server untuk alokasi IP dinamis ke klien, serta Web Server (LAMP Stack/Nginx) untuk hosting aplikasi web lokal.

Keamanan & Pemeliharaan: Penerapan aturan pengamanan firewall melalui iptables/UFW serta penanganan masalah (troubleshooting) konflik IP Address dan interferensi sinyal jaringan nirkabel.

Teknologi Backend & Server: Node.js, Ubuntu Server, Debian Server

Infrastruktur Jaringan: RouterOS MikroTik, WinBox, Cisco Packet Tracer

Alat Produktivitas & Kontrol Versi: Git, GitHub, VS Code, Google Colab
